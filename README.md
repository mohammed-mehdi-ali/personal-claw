# 🧠 Personal Claw (AI Assistant System)

A full-stack AI-powered assistant inspired by ChatGPT, designed to process user queries, store conversations, and generate intelligent responses.

## 🚀 Features
- Chat-based interface
- AI-powered responses (LLM integration)
- Conversation history storage
- Backend API for processing queries
- Scalable architecture

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Hono / Express
- Database: PostgreSQL
- AI: Ollama / LLM integration

## ⚙️ How it works
1. User enters a query
2. Frontend sends request to backend
3. Backend processes using AI model
4. Response returned and displayed
5. Conversation stored in database
